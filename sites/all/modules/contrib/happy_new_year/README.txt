Happy New Year!
https://www.drupal.org/project/happy_new_year
=============================================

INTRODUCTION
------------

A module with garland and snow decoration that creates
longlasting atmosphere of New Year for you and the users of
your website.

REQUIREMENTS
------------

This module has no requires

INSTALLATION
------------

Install as you would normally install a contributed Drupal module. Visit:
https://drupal.org/documentation/install/modules-themes/modules-7
for further information.

CONFIGURATION
-------------
Go to admin/config/media/happy_new_year and enable garland and snow.
